# Descripción del programa
El programa gestiona un archivo csv con datos de paises. Permite agregar paises, modificar datos, ordenarlos, filtrarlos y mostrar algunas estadisticas.
# Instrucciones de uso
1. Descargar o clonar repositorio en tu PC
2. Ejecutarlo desde la terminal en TPI_luciano_nicolas.py
